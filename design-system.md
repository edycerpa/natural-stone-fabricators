# Design System

_Este archivo sirve como la fuente de la verdad para las variables de diseño, basándose en la configuración de Tailwind y el archivo global de estilos (`src/styles/global.css`)._

## Typography
- **Sans-serif (`--font-sans`)**: "Montserrat", ui-sans-serif, system-ui, sans-serif
- **Serif (`--font-serif`)**: "Savoye LET", "Savoye", "Apple Chancery", cursive
